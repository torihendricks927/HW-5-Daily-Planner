# HW-5-Daily-Planner

## Daily Planner

https://torihendricks927.github.io/HW-5-Daily-Planner/

This website provides a daily planner that can be modified to fit the users schedule. It shows the past, present, and future hours by coor coding the sections. The middle boxes are open for the user to place any plans or events for that hour followed by saving it with the save button icon to the right. It will remain on the schedule even if the website is refreshed. It can also be deleted if needed. This website is useful for anyone who wants to have a more organized schedule.
